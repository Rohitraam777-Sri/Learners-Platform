# Learners Platform

Learners Platform is a simple front-end learning website designed to help beginners understand web development concepts. The platform provides structured learning levels such as Beginner, Intermediate, and Advanced.

🚀 Features
- Beginner to Advanced learning sections
- Clean and responsive UI
- Easy navigation for students
- Built using HTML, CSS, and JavaScript

🛠️ Technologies Used
- HTML
- CSS
- JavaScript
